LEARN HTML

HTML is an acronym for Hypertext Markup Language. It is used to markup web pages. It is also called a markup language. HTML is used to build the structure of web pages. It is like the skeletal of web pages with only bones and no flesh. Think of it as the bricklayer who erects the structure of a building using blocks, cement, mortar etc. The structure ususally looks unappealing to the eye because it lacks presentation(beauty) and interactivity provided by CSS and Javascript respectively.

LEARN CSS3

CSS stands for Cascading Style Sheet which adds presentation to web pages; hence, CSS is a styling language. CSS can best be compared to the mason who plasters the walls of a building, the painter who uses different color combinations to paint the walls of a building,the carpenter who adds ceiling to a building. In short, CSS can be related to all the beautiul things added to a building. In this case, it means CSS adds beauty/presentation to the structure (HTML) of web pages.

LEARN JAVASCRIPT

Javascript is a programming language that adds interactivity to web pages. It is the only programming language that is used on the client side (i.e to interact with the browser). Since HTML, and CSS are the bricklayer and artist respectively, JavaScript is the Wizard who adds interactivity to a completed building. The remote control of all the electronic devices in a building can said to be the JavaScript because with it, the occupant could turn on/off a TV set, regulate the temperature of the air condition in a room. Also, the power switch of fans and light bulbs could also be compared to Javascript. In the real sense, the test section of this application is a typical example of the application of JavaScript in mobile/web development, because a user is able to take a test and get the test result on the fly. And that is the interaction that JavaScript offers in mobile/web applications.
