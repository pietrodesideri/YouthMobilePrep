FREQUENTLY ASKED QUESTIONS

For beginners, it is important to start by learning HTML before getting into CSS and then JavaScript. While learning how to write some code, it is also important to be patient, as impatience is the reason why some learners quit thinking programming is not their calling. It is not secret that even experienced developers sometimes spend days trying to debug their code, but because of patience, they would usually find a solution.

Like I mentioned in the about page, it would be in your best interest to learn computer basics such as powering on/off a computer, working with folders and files, knowing some basic computer keyboard shortcuts. Another important skill that you need is research to be able to find solutions by surfing the internet and reading programming books. As a trainee at the UNESCO-Abuja Youthmobile, I saw how some trainees struggle to learn because of their lack of basic computer skills and inability to research despite the huge amount of time the facilitators invested in guiding them. This does not only apply to those willing to join the Youthmobile training, but also anyone interested in learning computer programming. Again, all of these require some patience, dedication, passion and hardwork to do.

A text editor or IDE(Integrated Development Environment) is a software for writing codes. In no particular order, some of these IDEs include but not limited to;

  1. Atom
  2. Visual Studio Code
  3. Sublime Text
  4. Brackets
  5. Notepad
  6. Notepad++

These are by far the most widely used text editors by most developers. It is a matter of preference what text editor you used but open the next question to see what IDE I recommed for you.

As mentioned in the previous question, what code editor you use is a matter of preference. However, as a beginner with zero knowldge in mobile/web applications development, I would recommed you use lightweight IDEs such as Sublime Text and Notepad or Notepad++.
