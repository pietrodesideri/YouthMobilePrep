JavaScript is a programming language which adds interactivity to web pages. Applications built with only HTML CSS are usually static meaning that a user is unable to interact with such applications. For instance, the menu that opens/closes when you touch the menu icons is a function of JavaScript. Also, when you take the test in this application, you will get a result instantly all thanks to JavaScript or what I love to call The Wizard.. It is important to note that JavaScript is not the same as same Java. They are entirely diffrently programming languages. The official name for JavaScript is EcmaScript.

JavaScript can be written in one of the following two ways;

  1. Embedded script within the HTML head or body tags
  2. External script in an external script file

To write JavaScript in the HTML document is called Embedded JavaScript. This can be done by creating the script tag either within the head or the body tags. As you get advance in learning JavaScript, you will learn why it is so.

Again, for seperation of concern, it is recommended to write JavaScript in an external script file. You do this by creating a file with the .js name extension and then link it to HTML document(s). In JavaScript and all other programming languages, programs are written using data. And these data are stored in containers called varaibles. These data include;

  1. number as in 1, 2.6, 3, 4....
  2. string as in "Hello"/'Hello'. Strings are wrapped in either double/single quotes.
  3. bolean as in true of false
  4. null
  5. undefined
  6. primitives (object and arrays).

As mentioned in the previous lesson, variables are containers for storing data. For instance, in basic arithmetic we could have x = 2. This means that x is the conatainer/variable while 2 is the data. It is similar to JavaScript; the only difference is the var/let/const keywords used before the variable name in JavaScript. The variable can hold any data type. In This tutorial, we will stick to the var key word

As you have learned in HTML and CSS lessons, comments are used by developers do know what is written where and sometimes why it is written. In javascript, we write comments as single line // or multi line /* */
